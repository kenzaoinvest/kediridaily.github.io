Rumpun Bambu di Pare Kediri Terbakar Akibat Api Pembakaran Sampah, Damkar Selamatkan Aset Rp500 Juta

Kediri – Kebakaran rumpun bambu terjadi di RT 02/RW 10, Dusun Jombangan Santren, Desa Tertek, Kecamatan Pare, Kabupaten Kediri, Kamis (23/7/2026) sore.

Beruntung, api yang sempat mengancam permukiman warga berhasil dipadamkan petugas Pemadam Kebakaran (Damkar) Kabupaten Kediri sebelum merembet lebih luas.

Peristiwa itu bermula dari laporan warga bernama Zulianto yang diterima petugas sekitar pukul 15.17 WIB. Tim Damkar Pos Grogol kemudian bergerak menuju lokasi pada pukul 15.20 WIB dan tiba sekitar pukul 16.10 WIB.

Berdasarkan hasil penanganan di lapangan, kebakaran diduga dipicu aktivitas pembakaran sampah. Tiupan angin yang cukup kencang membuat api membesar hingga menjalar ke rumpun bambu yang berada di dekat rumah warga. Luas area yang terbakar diperkirakan mencapai sekitar 10 ru.

Plt. Kepala Satuan Polisi Pamong Praja (Satpol PP) Kabupaten Kediri, Kaleb Untung Satrio Wicaksono, mengatakan petugas langsung melakukan pemadaman begitu tiba di lokasi agar api tidak merambat ke bangunan di sekitarnya.

"Begitu menerima laporan, tim segera menuju lokasi dan melakukan pemadaman sesuai prosedur. Api berhasil dikendalikan sehingga tidak sampai merembet ke rumah-rumah warga di sekitar lokasi," ujar Kaleb.

Dalam operasi tersebut, Damkar Kabupaten Kediri menerjunkan satu unit armada Pos Grogol berkapasitas 5.000 liter beserta empat personel untuk memadamkan kobaran api.

Kaleb mengimbau masyarakat agar lebih berhati-hati saat membakar sampah, terutama pada musim kemarau atau ketika angin bertiup kencang.

"Kami mengingatkan masyarakat agar tidak membakar sampah sembarangan. Pastikan api benar-benar padam dan jangan ditinggalkan, karena kondisi angin dapat membuat api cepat membesar dan memicu kebakaran," tambahnya.

Kebakaran berhasil dipadamkan sekitar pukul 18.29 WIB. Tidak ada korban jiwa maupun korban luka dalam kejadian tersebut.

Akibat peristiwa itu, kerugian materi diperkirakan mencapai sekitar Rp3 juta, sementara aset yang berhasil diselamatkan petugas ditaksir bernilai sekitar Rp500 juta.
